
# Git Mastery Badge Level 1

## Competencies

- Learn how to install git
- Learn how to make changes in the source code and ensure the changes is managed in Git:
	- Cloning
	- Adding
	- Committing
	- Pushing
	- Branching
	- Merging
	- Pull Requests
	-  How to cherry pick
- Learn to solve merge conflict issues
